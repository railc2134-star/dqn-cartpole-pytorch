# CartPole DQN Project

This project is an implementation of a Deep Q-Network (DQN) to solve the CartPole problem. The CartPole problem is a classic control problem where a pole is attached to a cart and the goal is to balance the pole by applying forces to the cart. The project uses the Gymnasium library to simulate the environment and PyTorch to implement the DQN.

## Technologies Used
- Gymnasium: A library for developing and comparing reinforcement learning algorithms.
- PyTorch: A library for deep learning.

## How to Run
1. Install the required libraries by running `pip install gymnasium torch numpy`.
2. Run the project by executing `python CartPole.py`.
3. Enter `1` to load a trained model or `2` to train a new model.
